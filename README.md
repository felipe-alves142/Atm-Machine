
# ATM Machine


## O que é 

   ATM Machine é um caixa 24 horas que permite fazer operações básicas na sua conta bancária.
   
   ### Funções 
   
   - Login 
   - Ver o extrato
   - Retirar uma quantia
   - Depositar uma quantia
   
## Instalação

- Clone o projeto na sua máquina
- Execute a classe main

## Técnologias 

- Java Core
- Intellij ( IDEA)
